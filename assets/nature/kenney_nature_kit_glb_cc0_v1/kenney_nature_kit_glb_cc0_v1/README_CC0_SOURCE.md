# Kenney Nature Kit - GLB Ready Pack

This package is a GLB-focused redistribution of the CC0 Nature Kit by Kenney.

It is intended for quick import into 3D tools, game engines, illustration workflows,
manga backgrounds, outdoor scenes, fantasy maps, gardens, forests, rivers, cliffs,
campsites, farms, and general scene dressing.

## Included

- `models_glb/` - 329 GLB nature and outdoor prop models
- `renders_png/` - 329 PNG reference renders, one per GLB model
- `MODEL_LIST.txt` - file list for the GLB models
- `Preview_Kenney.png` and `Sample_Kenney.png` - original overview images from the source package
- `License_Kenney.txt` - original license text included with the source package
